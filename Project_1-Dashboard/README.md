# Data Jobs Dashboard with Power BI

> A Power BI dashboard for exploring 2024 data science job trends, salary insights, and hiring patterns.

## Introduction

This dashboard was created for job seekers, job transitioners, and job swappers who want a clearer view of the data job market. Using a real-world dataset of 2024 data science job postings, it provides a simple and interactive way to explore titles, salaries, locations, and market trends.

### Dashboard File

You can find the Power BI file here: [`Data_Jobs_Dashboard.pbix`](Data_Jobs_Dashboard.pbix)

## Skills Showcased

This project demonstrates several Power BI skills, including:

- Data transformation with Power Query, including cleaning, shaping, handling blanks, and changing data types.
- Building measures and KPIs such as median yearly salary and job count.
- Using core visuals like column, bar, line, and area charts for comparisons and trend analysis.
- Applying map charts for geographic analysis.
- Designing cards and tables to highlight key metrics and detailed data.
- Creating a clean and effective dashboard layout.
- Adding interactivity with slicers, buttons, bookmarks, and drill-through navigation.

## Dashboard Overview

This report is split into two pages to provide both a summary view and a deeper analysis.

### Page 1: High-Level Market View

This page provides a quick overview of the data job market. It highlights key KPIs such as total job count, median salaries, and top job titles so users can understand the market at a glance.

### Page 2: Job Title Drill Through

This page provides a detailed breakdown for a single job title. It includes salary ranges, work-from-home statistics, top hiring platforms, and a global map of job locations.

## Conclusion

This dashboard shows how Power BI can transform raw job posting data into a useful career analysis tool. It helps users filter, explore, and drill into the data to make better career decisions.
